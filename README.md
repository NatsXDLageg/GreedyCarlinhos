# Greedy Carlinhos

Um jogo sobre um menino normal, que queria fazer um joguinho...

### Pré Requisitos

Para editar modificar o jogo: uma cópia do programa `RPG Maker VX Ace`.

## Instalação:

RPG Maker -> File -> Open Project -> `path/to/project`.
